使用最快速简单方式自定义View动态折线图

注意：此方法需要开发者具备一定的基础，不然将会一头雾水

一、创建自定义类

  • 创建类让此类继承于View

二、重写方法

​ 当类继承View时需要重写它里面的三个方法

​ 此时需要在视图窗口中画线条就需要重写onDraw方法

三、初始化

  • 对视图的操作全都在onDraw方法中实现

  • 首先先初始化笔的对象Paint 对笔进行初始化

    • 比如笔的颜色和是否抗锯齿等等

四、绘制X与Y轴

  • 使用onDraw里面参数提供的画布，画布里面有封装好的方法，我们只需要给它初始值的坐标即可

五、将数值刻到Y轴上

  • 将数值刻到Y轴上，画布也提供了相对应的canvas.drawText()方法

六、绘制动态线图

  • 先创建存储数据的容器

    • ~~~java
      double[] num = new double[6];

  • 绘制动态线

    • ~~~java
      /**
       * 画动态连接线
 * @param canvas
 */
private void drawLineTo(Canvas canvas) {
    int[] xs = {80,180,280,380,480,580};
    for (int i = 0; i < xs.length; i++) {
        if(i<5){
            paint.setColor(Color.RED);
            canvas.drawLine(xs[i], (float)((int)200-(num[i]/100)*100), xs[i+1], (float)(int)(200-(num[i+1]/100)*100),paint);
        }
    }
}
