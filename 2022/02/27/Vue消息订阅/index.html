Vue消息订阅

消息订阅与发布

  1. 订阅消息
  2. 发布消息

哪个组件需要消息就在哪个组件中订阅消息

使用

安装

引入一个外置的库pubsub.js 它可以实现消息的订阅与发布

安装方式:

npm install pubsub-js

引入

在订阅的组件中引用pubsub-js

import pubsub from 'pubsub-js'

订阅消息

在方法内部调用subscribe(消息名, 回调)方法

回调有两个参数第一个是方法名，第二个才是回调的数据

pubsub.subscribe('hello', function(func,data){函数体})

发布消息

1、在收取订阅信息的组件中先引入pubsub-js

import pubsub from 'pubsub-js'

2、收取信息

在方法中调用publish(‘订阅名’, 回调)

pubsub.publish('hello', 数据体)

销毁消息

当组件要被销毁的时候需要取消订阅

在销毁钩子的内容写销毁消息
