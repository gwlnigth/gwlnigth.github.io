Vue动画

概述

在插入、更新或者移除DOM元素时，在合适的地方添加样式类名

使用

  1. 准备好样式
    • 元素进入的样式
      1. v-enter : 进入的起点
      2. v-enter-active : 进入过程中
      3. v-enter-to : 进入的终点
    • 元素离开的样式
      1. v-leave : 离开的起点
      2. v-leave-active : 离开的过程
      3. v-leave-to : 离开的终点
  2. 使用<transition>包裹要过渡的元素，并配置name属性

注意:如果有多个元素需要过滤, 则需要使用<transition-group>, 且每一个元素都要指定key值
